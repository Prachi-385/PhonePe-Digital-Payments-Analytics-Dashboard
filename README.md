# PhonePe Digital Payments Analytics Dashboard

## 📌 Project Overview

The **PhonePe Digital Payments Analytics Dashboard** is an interactive Power BI project designed to analyze digital payment transactions and uncover insights related to user behavior, transaction performance, service usage, and business growth. The dashboard enables stakeholders to monitor key performance indicators (KPIs), identify trends, and make data-driven decisions.

---

## 🎯 Business Objective

The objective of this project is to analyze transaction data and answer critical business questions such as:

* How much transaction volume and value are being generated?
* What is the overall transaction success rate?
* Which payment services contribute the most transaction value?
* How do transaction patterns vary across months, days, and quarters?
* Which customer segments are the most active?
* How does transaction activity differ between weekdays and weekends?

---

## 📊 Dashboard Highlights

### Key Performance Indicators (KPIs)

* **Total Transaction Amount:** ₹378.19M
* **Total Transactions:** 150K
* **Average Transaction Value:** ₹2.52K
* **Transaction Success Rate:** 96.04%
* **Active Users:** 108K
* **MoM Transaction Growth:** 8.98%
* **MoM Amount Growth:** 9.04%

### Visualizations Included

* Monthly Transaction Trend Analysis
* Daily Transaction Trend Analysis
* Revenue by Service Type
* Transaction Distribution by Age Group
* Weekend vs Weekday Transaction Analysis
* Quarterly Success vs Failed Transactions
* Top 5 Customers by Transaction Amount

---

## 🔍 Key Insights

* Processed **150K transactions** worth **₹378.19M** with a strong **96.04% success rate**.
* **Gen X (38%)** and **Millennials (37%)** accounted for nearly **75% of total transactions**.
* Transaction activity was significantly higher on **weekdays (71.64%)** compared to weekends (28.36%).
* The platform recorded **108K active users** with an average transaction value of **₹2.52K**.
* Achieved positive growth with **8.98% MoM transaction growth** and **9.04% MoM transaction amount growth**.

---

## 💡 Recommendations

* Develop targeted campaigns for Millennials and Gen X users to improve engagement and retention.
* Introduce weekend-specific offers and cashback programs to increase weekend transaction volume.
* Investigate failed transactions by service category to further improve transaction reliability.
* Leverage high-value customers through loyalty and referral programs.
